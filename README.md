# 小区闲置交易 🏘️

> 基于微信小程序的社区闲置物品交易平台（微信云开发版）

## 项目简介

**小区闲置交易**是一款微信小程序，旨在方便同一社区居民之间发布、浏览和交易闲置物品。用户可以通过手机号登录，发布闲置商品（上传图片、填写描述、设定价格、标注位置），浏览附近邻居发布的商品，并通过内置聊天功能与卖家沟通。

## 功能概览

### 🔐 手机号登录
- 手机号 + 验证码登录方式
- 微信云函数自动获取 openid
- 用户信息存入云数据库

### 🏠 首页浏览
- 商品列表从云数据库实时加载
- **关键词搜索**商品
- **分类筛选**：全部、家电、家具、服装、书籍、玩具、其他
- 基于用户当前位置计算商品距离
- 下拉刷新

### 📝 发布 / 编辑商品
- 上传商品图片到**云存储**（最多 3 张）
- 填写商品名称、详情描述、价格
- 选择商品分类
- 获取并标记商品所在位置
- 支持编辑已发布的商品
- 商品数据通过云函数存入云数据库

### 📦 商品详情
- 查看商品完整信息
- 图片轮播展示
- 查看卖家信息
- 一键导航至商品所在位置
- 联系卖家（内置聊天）

### 💬 在线聊天
- 消息存入云数据库，支持多用户互通
- 与卖家/买家实时文字沟通
- 支持发送图片消息（云存储）
- 自动轮询新消息

### 👤 个人中心
- 编辑昵称和头像（头像通过微信头像组件选择）
- 查看"我发布的"商品列表（从云数据库）
- 商品上架 / 下架管理（云函数操作）
- 编辑或删除已发布商品
- 退出登录

## 技术栈

| 类别 | 技术 |
|------|------|
| 框架 | 微信小程序原生框架 |
| 语言 | JavaScript + WXML + WXSS |
| 后端 | **微信云开发（云函数 + 云数据库 + 云存储）** |
| 数据库 | 云数据库（3 个集合：users、goods、messages） |
| 存储 | 云存储（商品图片 + 聊天图片） |
| 定位服务 | 微信地图 API（wx.getLocation） |
| 距离算法 | Haversine 公式 |

## 云数据库设计

| 集合名 | 用途 | 关键字段 |
|--------|------|----------|
| `users` | 用户信息 | openid, nickName, avatarUrl, phone |
| `goods` | 商品信息 | openid, name, desc, price, category, cloudImageIds, latitude, longitude, status |
| `messages` | 聊天消息 | goodsId, fromOpenid, toOpenid, content, type, isRead |

## 快速开始

### 环境要求

- [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)（最新版）
- 微信小程序 AppID（需开通云开发）
- 微信云开发环境（在微信开发者工具中开通）

### 🚀 首次部署步骤

#### 1. 开通云开发

1. 在微信开发者工具中打开项目
2. 点击左上角「云开发」按钮
3. 按提示开通云开发环境，记录你的**环境 ID**
4. 将 `app.js` 和 `project.config.json` 中的环境 ID 替换为你的实际环境 ID

#### 2. 修改环境 ID

**`app.js` 第 9 行：**
```js
wx.cloud.init({
  env: '你的环境ID',  // 替换这里
  traceUser: true
})
```

#### 3. 部署云函数

在微信开发者工具中，右键点击 `cloudfunctions` 目录下的每个云函数文件夹（login、goods、chat、initDB），选择 **「上传并部署：云端安装依赖」**。

部署顺序建议：
1. `login`
2. `initDB`
3. `goods`
4. `chat`

#### 4. 初始化数据库

部署完成后，在开发者工具中调用 `initDB` 云函数来初始化数据库集合：

1. 在「云开发控制台」→「云函数」中找到 `initDB`
2. 点击「测试」运行即可
3. 或者也可以在首次使用时自动创建（集合会在第一次写入数据时自动建立）

#### 5. 设置数据库权限

在「云开发控制台」→「数据库」中，为每个集合设置权限：

| 集合 | 权限设置 | 说明 |
|------|----------|------|
| `users` | 仅创建者可读写 | 用户只能修改自己的信息 |
| `goods` | 所有用户可读，仅创建者可写 | 所有人可浏览，只有发布者可修改 |
| `messages` | 所有用户可读，仅创建者可写 | 聊天双方可读写自己的消息 |

#### 6. 运行预览

点击「编译」按钮运行，或点击「预览」生成二维码在手机上体验。

## 使用说明

### 登录
首次进入需使用手机号 + 验证码登录（当前验证码为演示模式，输入任意 11 位手机号即可）。

### 发布商品
1. 点击底部 TabBar「发布」
2. 上传商品图片（至少 1 张，最多 3 张）→ 上传到云存储
3. 填写商品名称、详情描述、价格
4. 选择商品分类
5. 点击获取当前位置
6. 点击「发布闲置」提交 → 调用 goods 云函数写入数据库

### 管理商品
1. 进入「我的」→「我发布的」
2. 可对商品进行**编辑**、**上架/下架**、**删除**操作（均通过云函数操作）

### 联系卖家
在商品详情页点击「联系卖家」，进入聊天页面。消息通过 chat 云函数存入云数据库，支持文字和图片消息。

## 云函数说明

| 云函数 | 操作(action) | 功能 |
|--------|-------------|------|
| `login` | - | 获取 openid，创建/返回用户信息 |
| `goods` | `publish` | 发布商品 |
| | `list` | 商品列表（搜索+分类+分页） |
| | `detail` | 商品详情 |
| | `update` | 编辑商品 |
| | `delete` | 删除商品 |
| | `toggleStatus` | 上下架切换 |
| | `myList` | 我的商品列表 |
| `chat` | `send` | 发送消息 |
| | `getMessages` | 获取会话消息 |
| | `getConversations` | 获取会话列表 |
| `initDB` | - | 初始化数据库集合 |
